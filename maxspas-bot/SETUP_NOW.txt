MAXSPAS Studio — 3 шага прямо сейчас в браузере
=================================================

1. Откройте бота: https://t.me/maxspas_studio_bot
   Нажмите «Start» (или отправьте /start).

2. В терминале (папка maxspas-bot) запустите:
   python get_admin_id.py
   Скрипт запишет ваш ADMIN_CHAT_ID в .env.

3. Запустите бота:
   python bot.py
   (или .\run_bot.ps1)

После этого проверьте сайт: Контакты → форма → «Отправить в Telegram».
