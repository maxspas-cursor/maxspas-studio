# MAXSPAS Studio Telegram Bot

Бот для приёма заявок и уведомлений администратора. Сайт MAXSPAS Studio отправляет лиды через [deep link](https://t.me/) в личный Telegram (`js/layout.js`); этот бот работает отдельно — когда пользователь пишет **боту** напрямую.

## Возможности

- `/start` — приветствие и кратко об услугах MAXSPAS Studio
- `/help` — справка по командам
- Любое сообщение пользователя пересылается администратору (`ADMIN_CHAT_ID`)

## Быстрый старт

### 1. Python 3.10+

Убедитесь, что установлен Python 3.10 или новее.

### 2. Настройка окружения

```powershell
cd maxspas-bot
copy .env.example .env
```

Отредактируйте `.env`:

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен от [@BotFather](https://t.me/BotFather) |
| `ADMIN_CHAT_ID` | Ваш числовой chat id |

### 3. Как узнать ADMIN_CHAT_ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram.
2. Нажмите **Start** / отправьте любое сообщение.
3. Бот пришлёт ваш **Id** — это число, например `123456789`.
4. Вставьте его в `.env`: `ADMIN_CHAT_ID=123456789`

Для группы: добавьте бота в группу, напишите сообщение и используйте id группы (часто отрицательное число). Удобнее сначала настроить личный чат.

### 4. Запуск

**PowerShell (рекомендуется):**

```powershell
.\run_bot.ps1
```

**Вручную:**

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python bot.py
```

Бот работает, пока открыт терминал. Для продакшена используйте VPS, systemd или хостинг с постоянным процессом.

## Связь с сайтом

На сайте (`js/layout.js`, `js/main.js`) форма контактов открывает:

```
https://t.me/Maxspas?text=...
```

с готовым текстом заявки. Это **личный** Telegram, не API бота.

Этот репозиторийный бот нужен, если вы:

- даёте ссылку `https://t.me/<имя_вашего_бота>` на сайте или в рекламе;
- хотите получать сообщения, отправленные **боту**, в один админ-чат.

Чтобы заявки с сайта шли в бот автоматически, потребуется отдельная интеграция (webhook/API на сервере). Сейчас сайт и бот связаны логически, но не через единый backend.

## Структура

```
maxspas-bot/
  bot.py           # основной код (aiogram 3.x)
  requirements.txt
  .env.example     # шаблон (коммитится)
  .env             # секреты (НЕ коммитить)
  run_bot.ps1      # локальный запуск
```

## Безопасность

> **Важно:** токен бота был передан в открытом чате. После настройки **обязательно перевыпустите токен** в [@BotFather](https://t.me/BotFather):
>
> 1. Откройте своего бота в BotFather.
> 2. Команда `/revoke` (или **API Token → Revoke current token**).
> 3. Скопируйте новый токен в `.env`.
>
> Никогда не коммитьте `.env` и не публикуйте токен в репозитории, скриншотах или чатах.

Файл `.env` добавлен в `.gitignore` корня проекта.
